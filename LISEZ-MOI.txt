  SUIVI DE LECTURE ZOTERO
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Cet outil affiche tous vos articles Zotero dans un tableau de bord
dans votre navigateur. Il se connecte à Zotero en temps réel et
se met à jour automatiquement dès que vous travaillez dans Zotero.


╔══════════════════════════════════════╗
║  CE QUE FAIT L'OUTIL                 ║
╚══════════════════════════════════════╝

L'outil détecte automatiquement votre progression de lecture :

  ✅ Lu        → vous avez ajouté le tag "lu" dans Zotero,
                 ou vous avez cliqué sur le statut dans le tableau de bord

  📖 En cours  → le PDF contient des annotations

  👁 Consulté  → le PDF a été ouvert mais pas encore annoté
                 (Mac uniquement — voir note Windows ci-dessous)

  ⬜ À lire    → document jamais ouvert

Autres fonctionnalités :
  • Articles organisés par collection et sous-collection Zotero
  • Barre de progression par article (page atteinte / total pages)
  • Filtres par statut dans chaque collection
  • Recherche par titre ou auteur
  • Bouton "Tout marquer Lu" par collection
  • Mise à jour en direct (point vert = connecté)
  • 📝 Synthèse : compilation de toutes vos annotations par collection


╔══════════════════════════════════════╗
║  VOUS ÊTES SUR MAC ?                 ║
╚══════════════════════════════════════╝

Étape 1 — Sur GitHub, cliquez sur le bouton vert "Code"
          puis "Download ZIP" et décompressez le dossier
Étape 2 — Double-cliquez sur "Démarrer le suivi.command"
          (La première fois : clic droit → Ouvrir → Ouvrir)
Étape 3 — Une fenêtre noire s'ouvre, laissez-la ouverte
Étape 4 — Votre navigateur s'ouvre automatiquement

C'est tout !


╔══════════════════════════════════════╗
║  VOUS ÊTES SUR WINDOWS ?             ║
╚══════════════════════════════════════╝

⚠ Avant la première utilisation, installez Python.
  C'est un logiciel gratuit, comme Adobe Reader :
  vous le faites une fois, et c'est terminé.

─── Installation de Python (à faire une seule fois) ───────────

  1. Allez sur :  https://www.python.org/downloads
  2. Cliquez sur le gros bouton "Download Python"
  3. Ouvrez le fichier téléchargé
  4. IMPORTANT : cochez "Add Python to PATH"
     (tout en bas de la fenêtre d'installation)
  5. Cliquez sur "Install Now" et attendez la fin

─── Utilisation ────────────────────────────────────────────────

  1. Sur GitHub, cliquez sur le bouton vert "Code"
     puis "Download ZIP" et décompressez le dossier
  2. Double-cliquez sur "Démarrer le suivi.bat"
  3. Une fenêtre noire s'ouvre, laissez-la ouverte
  4. Dans Chrome ou Edge, tapez :  http://localhost:7777

  ⚠ Note Windows : la détection des documents "Consultés"
    (ouverts sans annotation) n'est pas disponible sur Windows.
    Ces documents resteront en "À lire" jusqu'à ce que vous les
    annotiez ou changiez leur statut manuellement en cliquant.


╔══════════════════════════════════════╗
║  MODIFIER UN STATUT MANUELLEMENT     ║
╚══════════════════════════════════════╝

Cliquez sur le bouton de statut d'un article pour le changer :

  ⬜ À lire  →  👁 Consulté  →  📖 En cours  →  ✅ Lu

Les changements sont sauvegardés automatiquement.


╔══════════════════════════════════════╗
║  PRÉREQUIS                           ║
╚══════════════════════════════════════╝

  • Zotero installé sur votre ordinateur
  • Python 3 (déjà présent sur Mac — à installer sur Windows)
