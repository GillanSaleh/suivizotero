# suivizotero
Tableau de bord local pour suivre sa progression de lecture dans Zotero
Suivi de lecture Zotero
Un outil simple pour visualiser et suivre votre progression de lecture directement depuis votre bibliothèque Zotero.

Ce que ça fait
Affiche tous vos articles organisés par collection
Détecte automatiquement les articles lus (grâce à vos annotations PDF)
Permet de marquer un article "En cours" ou "À lire" en un clic
Se met à jour en temps réel dès que vous travaillez dans Zotero

Installation
Téléchargez les fichiers
Installez Python si ce n'est pas déjà fait → python.org/downloads (Windows uniquement — cochez "Add Python to PATH" pendant l'installation)
Double-cliquez sur Démarrer le suivi.command (Mac) ou Démarrer le suivi.bat (Windows)
Ouvrez votre navigateur à l'adresse : http://localhost:7777

Prérequis
Zotero installé sur votre ordinateur
Python 3 (déjà présent sur Mac, à installer sur Windows)
